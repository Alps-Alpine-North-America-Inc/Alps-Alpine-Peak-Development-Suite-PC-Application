# PEAK PC Application

## Beta Disclaimer

This version of the PEAK PC Application is an internal beta release only. It is not a final product and should not be treated as production-ready software. This internal beta version is provided as-is for testing and development purposes only. It is not intended for customer release, production deployment, or final validation.

This application is intended for Windows PC use only. It is used to communicate with, configure, and live-view supported PEAK evaluation board hardware and connected peripherals during development and internal testing.

## Current Status

- Internal release
- Beta version
- Windows PC application only
- Not final product
- Subject to UI, workflow, protocol, and feature changes
- Intended for development, validation, and internal feedback

## Important Notes

This software may contain incomplete features, known issues, temporary workflows, and debugging behavior. Some functionality may change or be removed before a final release.

Do not distribute this version externally unless explicitly approved by the project owner or development team.

## Intended Use

Use this application to:

- Connect to supported PEAK Development hardware
- Read and write board configuration data
- Configure connected peripherals
- View live peripheral data
- Test rules, actions, haptic waveforms, and related board behavior
